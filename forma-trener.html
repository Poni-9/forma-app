Profil, Gemini API in povezava s Stravo

👤 Tvoj profil

✨ Gemini API

Brezplačen ključ dobiš na aistudio.google.com → Get API key. Ključ ostane samo v tvojem brskalniku.

🟠 Strava povezava

Kako se povežem? (enkratna nastavitev, ~5 min)
  1. Pojdi na strava.com/settings/api in ustvari aplikacijo (ime poljubno, kategorija "Training").
  2. Pod Authorization Callback Domain vpiši localhost.
  3. Prepiši Client ID in Client Secret spodaj in klikni Poveži.
  4. Odpre se Strava — potrdi dostop. Preusmerilo te bo na "localhost" (stran se ne odpre — to je v redu!). Iz naslovne vrstice skopiraj cel naslov (ali samo kodo za code=) in jo prilepi nazaj sem.

Pozor: Strava od junija 2026 za API dostop zahteva Strava naročnino. Iz brskalnika gre prek javnega proxyja (CORS). Če nimaš naročnine, uporabi Intervals.icu spodaj — je brezplačen.

📈 Intervals.icu brezplačna alternativa

Kako se povežem? (~2 min, brezplačno)
  1. Registriraj se na intervals.icu in poveži svojo Stravo/Garmin (vožnje se sinhronizirajo same).
  2. Pojdi v Settings → čisto spodaj Developer Settings → API Key (generiraj ključ).
  3. Athlete ID najdeš v naslovu strani, npr. intervals.icu/athlete/i12345 → vpiši i12345.

💾 Podatki

Enkratno: brezplačen račun na jsonbin.io → API Keys → Master Key prilepi sem. Kodo, ki jo dobiš, prilepi še v aplikacijo KalorijeAI na telefonu — plan in spremembe se potem prenašajo v obe smeri.
